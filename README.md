# YaTray

Маленькое приложение для macOS: иконка в меню-баре (трее), быстрый запуск Яндекс Музыки и перехват медиа-клавиши Play. Нажал Play - открывается Музыка, а не iTunes.

Иконка YaTray

---

## Что умеет

- **Иконка в меню-баре** - компактная иконка плей в стиле системных иконок macOS. Не занимает место в доке.
- **Левый клик** по иконке - показать или скрыть окно Яндекс Музыки (если уже запущена).
- **Правый клик** - меню:
  - Показать окно / Запустить Яндекс Музыку - в зависимости от того, запущена ли Музыка.
  - Автозапуск - включить/выключить запуск YaTray при входе в систему.
  - Страница проекта - открыть этот репозиторий в браузере.
  - Версия - номер сборки (год.месяц.день.часминута).
  - Выйти - закрыть YaTray (⌘Q).

Меню YaTray в трее

- **Медиа-клавиша Play** (или F8): если Яндекс Музыка не запущена - YaTray её запустит и перехватит событие, чтобы не открывался iTunes. Если уже запущена - выведет окно на передний план. Для этого нужно один раз выдать YaTray разрешение "Управление компьютером" (Accessibility) в Системных настройках.

Иконка плей в трее

---

## Зачем это нужно

Удобно держать доступ к Яндекс Музыке из трея и не искать приложение в доке. Нажал Play - открылась Музыка. Никакого лишнего функционала.

## Установка (готовая сборка)

1. Открой страницу релизов: [Releases · lyucean/YaTray](https://github.com/lyucean/YaTray/releases).
2. Скачай последний релиз - файл `YaTray-macOS-XXXXXXXX.dmg` (в блоке Assets). Не качай Source code - там исходники.
3. Открой скачанный DMG (двойной клик по файлу).
4. В открывшемся окне перетащи YaTray в папку Программы (Applications).

Окно установки DMG

5. Готово. YaTray появится в меню-баре после первого запуска (найди в Программы и запусти).

На Mac должна быть установлена [Яндекс Музыка](https://music.yandex.ru/) с официального сайта. YaTray только запускает это приложение и перехватывает Play.

## Сборка из исходников

Если хочешь собрать сам (поковырять код или не доверять готовому DMG):

### Что нужно

- macOS (желательно последняя версия).
- Xcode с поддержкой Swift (достаточно Command Line Tools, но проще с полным Xcode).
- Установленная Яндекс Музыка в `/Applications` (иначе "Запустить Музыку" ничего не откроет).

### Шаги

1. Клонируй репозиторий:
   ```bash
   git clone https://github.com/lyucean/YaTray.git
   cd YaTray
   ```
2. Собери проект:
   ```bash
   make build
   ```
   Готовое приложение будет в `build/Build/Products/Release/YaTray.app`.
3. Запустить без установки:
   ```bash
   make run
   ```
4. Установить в Программы:
   ```bash
   make install
   ```
5. Создать DMG-установщик (как в релизах):
   ```bash
   make dmg
   ```
   Файл появится в папке `release/`.

Другие команды: `make help` - покажет все цели (clean, zip, release и т.д.).
